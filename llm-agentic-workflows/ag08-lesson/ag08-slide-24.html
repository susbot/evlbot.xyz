Test your understanding of cascading failures in multi-agent systems.
1. A retrieval agent returns 5% of its expected records due to an API timeout that returned a partial success. The analysis agent processes the truncated data and produces confident-looking output. No error signal is emitted at any stage. Which failure pattern does this most closely represent?
2. A vector database used by 4 retrieval agents has a 12-second outage. All 4 agents retry simultaneously every 2 seconds with no jitter. The database recovers at 12 seconds but is immediately overwhelmed by the simultaneous retry bursts and fails again. This repeats for 4 minutes. Which mitigation would most directly have prevented the extended outage?
