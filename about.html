About

Hey, I'm Susbot.

I built evlbot.xyz because I couldn't find the kind of security content that worked for how I learn. Not walls of text. Not hour-long videos. Just clear, visual, slide-by-slide breakdowns of concepts that actually matter.

I'm a different kind of learner — and I figured if I needed this, someone else did too. So I turned the official OWASP documentation into interactive courses anyone can follow, for free.

This is an independent project. No company behind it, no funding — just curiosity and a lot of late nights. If it helps you, that's enough.
